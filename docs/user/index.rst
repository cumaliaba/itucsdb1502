User Guide
==========

The web application provides two user roles: *normal user role* and *admin user role*

*normal user role* is a visitor who is not signed in.

*admin user role* is a visitor who is signed in.

We have different templates and permissions for these roles. 

*Normal user* only views and navigates around pages.

*Admin user* can views, navigates and edit table values.
